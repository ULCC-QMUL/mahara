Elasticsearch Analyzer

Performs the analysis process of the search database.

The Maraha system uses the built-in mahara_analyzer.

if you wish to use a different analyzer here are instructions on making a custom analyzer http://www.elasticsearch.org/guide/reference/index-modules/analysis/custom-analyzer/

This setting cannot be changed from the web interface. You may change it by adding a line to your config.php setting the value of $cfg->plugin->search->elasticsearch->analyzer.

It is recommended that you don't change from the default "mahara_analyzer" unless you know what you are doing
